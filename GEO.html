• Home

1. Unter Configurations Browse anklicken

2. "New Dashboard" anklicken

3. "Add panel" auswählen

4. Die Leiste "Time series" auswählen

5. In Suchleiste "Geomap" eingeben

6. Geomap auswählen

7. Den Punkt "Data source" anklicken

8. "Infinity" auswählen

9. URL auswählen

10. API Link eingeben

11. "Table view" auswählen

12. Nun kann man die Datenpunkte einsehen

13. Unter Add Columns, werden neue Zeilen erstellt

14. Im Selector wird die Zeile mit den Datenwerten ausgewählt, in dem Beispiel(gauge)

15. Nun muss das Format ausgewählt werden

16. In diesem Fall nutzen wir Nummern

17. Die Zeilen für die lattitude und longitute Punkte erstellen

18. "Map view" auswählen, um die Map für die Datenpunkte richtig eintzustellen

19. "View" und "Zoom" werden angeziegt

20. Im Punkt "View" Coordinates auswählen

21. "Data layer" anklicken

22. Nun sieht man die Felder Latitude und Longitude

23. Im Feld latitude die Zeile lat auswählen

24. Im Feld longitude die Zeile long auswählen

25. Die Punkte können nun in der Map eingesehen werden

26. Im Feld "Color" die erstellt Zeile gauge auswählen

27. Die Punkte haben nun je nach Wert bestimmte Farben

28. Im Punkt Thresholds können nun die Schwellenwerte der Farben eingestellt werden

29. Unter dem Punkt "+ Add threshold" können neue Schwellenwerte hinzugefügt werden

30. Man hat in diesem Beispiel, nach dem Einfügen, 3 Schwellenwerte

31. Die Veränderung ist sofort auf Karte sofort erkennbar

32. Durch Klicken auf die jeweiligen Kreise können die Farben ausgewählt werden

33. Falls einem die Ansicht nicht passen sollte, kann dies unter Map view wiedeer angepasst werden.
