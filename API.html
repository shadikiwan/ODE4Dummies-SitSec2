API Abkürzung: Was bedeutet API?

API ist eine gebräuchliche, technische Abkürzung, die für Application Programming Interface steht. Ins Deutsche übersetzt bedeutet der Begriff API so viel wie Programmierschnittstelle.

API Definition: Was ist eine API?

Eine API ist ein Programmteil, welches von einem Softwaresystem zur Anbindung an das System anderen Programmen zur Verfügung gestellt wird. Diese definiert die Programmanbindung auf Quelltext-Ebene. Um eine solche Schnittstelle zur Verfügung zu stellen bedarf es einer umfassenden und präzisen Dokumentation derer Funktionen und Parametern. (wikipedia).

Wie funktioniert die API in unserem Fall?

Als Input müssen die Parameter festgeleget werden, bevor eine API aufgeruft wird, da diese Parameter die erforderlichen Informationen an die API übergeben, damit sie ihre Funktion ausführen kann. Der Default Output ist JSON, allerdings kann das Eingabeformat auch angeben werden, indem beim API-Aufruf eine Erweiterung hinzugefügt werden kann. Dies kann .json, .xml oder .txt sein. JSON ermöglicht Organisation der Daten und kann schnell und unkompliziert ausgelesen werden.
